# 감정일기 앱 — 경쟁사 리서치 리포트

> 조사일: 2026-05-09
> 대상: AI 기반 감정/상황 입력 → 심리 분석 모바일/웹 앱
> 경쟁 서비스: **Daylio**, **Reflectly AI**, **Moodfit**

## 1. 핵심 가치 제안 (Value Proposition)

| 서비스 | 한 줄 제안 | 핵심 컨셉 |
|---|---|---|
| **Daylio** | "한 글자도 쓰지 않고 하루를 기록하라" — 무드 다이어리 & 해피니스 트래커 | 감정 이모지 + 활동 태그를 2탭으로 기록 → 통계·상관관계 시각화 |
| **Reflectly AI** | "세계 최초의 인공지능 저널" — 개인 멘탈 헬스 동반자 | 긍정심리학·CBT·마음챙김 기반 AI가 매일 질문을 던지고 텍스트 일기를 구조화 |

### 2-3. UX 특징

| 영역 | Daylio | Reflectly AI | Moodfit |

| 첫 인상 톤 | 친근한 이모지 + 파스텔, "심플함" 강조 | 일러스트 + 부드러운 컬러, "AI 친구" 느낌 | 정보량 많음, 도구 카드형 그리드 — 클리닉컬 |

| 학습 곡선 | 매우 낮음 | 낮음 (앱이 가이드) | **높음** — 도구가 많아 어디서 시작할지 헷갈림 |
| 강점 UX | 빠른 진입, "Year in Pixels" 시그니처 시각화 | 매일 다른 AI 질문 → 글쓰기 트리거 자연스러움 | 멘탈 헬스 평가(PHQ-9 등)·치료사 공유 워크플로우 |
| 약점 UX | 글쓰기 일기는 부수적 — 분석/해석은 사용자 몫 | 결제·로그인 강제 시점 불명확, 한국어 X | 도구가 분리돼 있어 "한 화면에서 끝" 경험 부족 |

## 3. 내 프로젝트의 차별화 포인트 (내가 더 잘 할 수 있는 3가지)

### 차별화 ①. **자연어 한 번 입력 → AI 심리 분석 즉시 리턴**
- Daylio는 "기록만", Moodfit은 "도구 모음", Reflectly는 "질문은 AI지만 분석 깊이는 얕음".
- 우리 앱은 "지금 감정 + 그렇게 느낀 상황"을 **단일 자연어 입력**으로 받고, LLM이 즉시
  - **인지 왜곡 감지** (흑백논리, 과잉일반화, 자기비난 등)
  - **감정-상황 트리거 매핑**
  - **다음 질문/리프레임 제안**
  을 카드 형태로 응답 — "기록 → 통계"가 아니라 **"기록 → 통찰"**로 닫힌 루프.

### 차별화 ②. **한국어 정서·문화 맥락에 최적화**
- 경쟁 3사 모두 영어권 모델. Reflectly 리뷰에는 *"한국어 추가해주실수 있나요?"* 가 76명 추천을 받음.
- 우리 앱은 **눈치·체면·화병·가족/직장 위계** 같은 한국 특유의 스트레스 컨텍스트를 프롬프트/파인튜닝에 반영하고, UI 마이크로카피·예시 질문도 처음부터 한국어 네이티브.
- LLM 기반이라 "팀장이 자꾸 회식 강요해서 힘들어" 같은 비정형 한국어 상황도 그대로 분석.

### 차별화 ③. **ABC(상황-감정-생각) 구조의 단일 화면 입력**
- Daylio: 무드+활동 태그(표면), Moodfit: 도구마다 화면 다름(분산), Reflectly: 챗봇 형식(시간 소요).
- 우리는 CBT의 ABC 모델을 **한 입력 폼**에 녹여 자연어로 받고, 백엔드 LLM이
  1. **A**ctivating event(상황),
  2. **B**elief(해석/생각),
  3. **C**onsequence(감정/행동)
  을 자동 추출 → 시간이 지나면 "이런 상황에서 자주 이런 해석을 하시네요" 같은 **개인화된 메타 패턴 리포트** 제공.
- Daylio의 "Year in Pixels" 같은 시그니처 시각화의 텍스트/AI 버전 — **"This Month in Thought Patterns"** 같은 자기 성장 시각화도 가능.

## 4. AUDIENCES.md 인풋 — 경쟁 서비스 유저들의 불만 (Pain Points)

> 출처: Google Play Store 리뷰(한국·영어), 각 사 공식 사이트, 일반 리뷰 패턴.
> 이 항목은 추후 `AUDIENCES.md`로 분리해 페르소나 작성 시 인용.

### 4-1. Reflectly AI 유저 불만
| 카테고리 | 인용/요지 | 시사점 |
|---|---|---|
| 안정성/버그 | "한 달간 잘 쓰고 있었는데 갑자기 오류가 너무 많이 떠요. 글 작성창에서 스크롤 안 내려감, 수정한 내용 저장 안됨, 테마 초기화" (★3, 2020) | **데이터 안정성/저장**이 신뢰의 핵심 — 우리 앱은 자동 저장 + 로컬 캐시 필수 |
| 데이터 손실 | "처음 깔고 30분동안 일기 쓴 뒤 잠깐 나가버렸더니 다 사라지고 튜토리얼 처음부터" (★2, 2024) | 온보딩 중에도 입력값 보존, 익명/게스트 입력 후 계정 연동 가능해야 함 |
| 한국어 미지원 | "한국어 추가해주실 수 있나요? 이 어플보다 좋은 게 없어서 파파고로 캡쳐 번역해서 씀" (★5, 76명 공감) | **한국어 네이티브**가 즉각적 차별화 |
| 가격 불투명 | 홈페이지에 가격 미공개 — 앱 깐 뒤에야 결제 화면 노출 | 랜딩에서 **명확한 무료/유료 구분 표기** |
| 락인 | 클라우드 전용, 데이터 이전 어려움 | **CSV/JSON 내보내기 기본 무료 제공** |

### 4-2. Daylio 유저 불만
| 카테고리 | 요지 | 시사점 |
|---|---|---|
| AI/해석 부재 | 통계와 차트는 풍부하지만 "이게 무슨 뜻인지"는 사용자가 직접 해석해야 함 | **통계 + 자연어 해설**을 같이 제공 (예: "이번 주 무드가 낮았어요. 공통 키워드는 '회사', '잠 못 잠'이에요") |
| 글쓰기 일기는 부수적 | 메모 첨부 수준이라 텍스트 일기파에는 부족 | **텍스트 우선** 입력 + 무드 보조 구조로 차별화 |
| 무료 플랜 제한 | 무드 슬롯·목표·자동 백업·PIN(iOS) 등 핵심 기능이 Premium에만 | 무료 플랜에 **PIN/자동 저장** 같은 기본 보안은 풀어두기 |
| AI 분석 없음 | 패턴 발견은 100% 사용자 몫 | LLM이 패턴 자동 발견 — **"숨은 트리거 발견" 알림** |

### 4-3. Moodfit 유저 불만
| 카테고리 | 요지 | 시사점 |
|---|---|---|
| 학습 곡선 | 도구가 많아 (Mood Journal, Nervous System, CBT, Breathwork, Mindfulness, PHQ-9 등) 어디서 시작할지 어려움 | **단일 입력 → 자동 분기**: 한 번 적으면 LLM이 적절한 후속 도구를 *추천* |
| 기능 분산 | 도구가 카드별로 흩어져 "한 흐름"이 부족 | 우리 앱은 **글 한 편 = 분석 + 회고 + 다음 행동 제안** 한 화면 |
| 영어 전용 | 한국어 미지원 — 한국어 멘탈 헬스 시장 비어있음 | 한국어 네이티브 + 한국 정서 데이터셋 |
| 가격 진입장벽 | Premium $35.99/년 — 학생/청년층에게 부담 | **무료 + 합리적 한국 시장 가격**(예: 월 4,900원 / 연 39,000원 수준) |
| Insight 깊이 | 차트는 있지만 "이번 주 X가 무드를 낮췄어요" 식 자연어 인사이트는 약함 | LLM 기반 자연어 위클리 리포트 |

### 4-4. 공통 페인 → 우리가 잡을 기회
1. **"기록은 했는데 그래서 뭐?"** — 3사 모두 통계/일기를 보여주지만 *해석*은 사용자 몫. → **AI 해석/리프레임이 핵심 차별화**.
2. **한국어 멘탈 헬스 앱 시장의 빈 공백** — 글로벌 빅3가 한국어 미지원/문화 미반영.
3. **데이터 안정성·내보내기에 대한 불신** — 무료 CSV/JSON 백업, 로컬 우선 저장이 신뢰의 베이스.
4. **무료 vs 유료 경계 모호** — 명확한 가격·기능 비교 페이지를 랜딩에 두면 신뢰 ↑.
